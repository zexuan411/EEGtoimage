# Masked EEG Pretraining and Joint Tri-Modal Alignment for Visual Decoding

## Contributions 
1. Tri-modal alignment: We introduce a text–image–EEG multimodal alignment framework that uses textual descriptions (generated by a large language model) to jointly align modalities, improving semantic grounding of EEG embeddings and recognition performance.  
2. EEG encoder design: We propose an EEG encoder that combines a subject-specific layer to capture individual differences, graph-based channel-interaction modeling, and an iTransformer-based temporal encoder to capture long-range temporal dependencies.  
3. MAE-style pretraining: We adopt an MAE-like masked reconstruction pretraining for EEG (encoder–decoder reconstructing masked signals). The pretrained encoder is partially reused for downstream tasks to improve EEG representation learning and accuracy.

## Datasets
We use the Things initiative datasets:
- ThingsEEG and ThingsMEG — official page: https://things-initiative.org/#datasets

Place the preprocessed EEG data in:
```
./Things-EEG2/Preprocessed_data_250Hz/
```

## Quick start
1. Prepare environment
2. Download ThingsEEG/ThingsMEG data and put preprocessed EEG under:
   - `./Things-EEG2/Preprocessed_data_250Hz/`
3. Download pretrained checkpoints (link above) and place them in `./checkpoints/`.
4. Run the provided train scripts
